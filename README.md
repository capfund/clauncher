# CLauncher

<p align="center">
  <img src="./src/assets/clauncher-logo.png" height=100/>
</p>

A lightweight open-source Minecraft launcher written in Python using CustomTkinter
Supports multiple installations, profiles, version selection, and automatic downloading of game files, libraries, natives, and assets!

---

## Feature set

- Launch Minecraft from a clean, and modern GUI
- Automatically downloads required files
- Security features and verification of integrity
- Live game logs inside the launcher
- Light/Dark mode toggle
- Version selection (all official releases; modded MC is to be done :>)

---

## Requirements and Dependencies
- Python 3.10/3.11/3.12
- Java (must be in PATH!!)
- Internet connection (for downloading and first launch)
- and customtkinter, Pillow, requests, urllib3
